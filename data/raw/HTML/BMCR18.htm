  Error code : BMCR18 Alarm Message: Sealing Station Unlock Fail 

Alarm Description
Sealing Station locking cylinder failed to open completely when machine was stoped.

Reason & Remedy

1. The sealing unit locking cylinders are not operated. The read switches are not operated for the sealing unit lock confirmation.
    a. Check and confirm the read switch is on.
    b. Check and confirm no obstacle in the cylinder movement and correct alignment.

Seal Top Reed Switch. Note When sealing station is open Reed SW-1 should be on.


Seal Top Reed Switch. Note When sealing station is open Reed SW-3 should be on.


Seal Top Reed Switch. Note When sealing station is open Reed SW-5 should be on.
