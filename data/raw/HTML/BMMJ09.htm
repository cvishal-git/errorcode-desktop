  Error code : BMMJ09 Alarm Message: Print Mark Missing 

Alarm Description
This alarm occurs if print mark is not sensed by PRC sensor for 5 consecutive cycles.

Reason & Remedy

1. The eye mark is shifted left or right while the machine is running.
    a. Correct film and foil path to sense the eye mark by the scanner on every cycle.
    b. Teach PRC scanner. ? Refer Chapter "Maintenance" > Section "Setting the Print Registration Control Scanner"
2. The distance between the two eye marks on the lidding foil is not as per the format drawing.
    a. Check the distance between two eye marks on the lidding foil
    b. Set the correct distance between two eye marks on the lidding foil if needed.
3. The entered PRC value is not correct for the appropriate PRC correction.
    a. Enter accurate PRC values to maintain correct PRC correction.
