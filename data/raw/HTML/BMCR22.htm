  Error code : BMCR22 Alarm Message: Embossing Low Temperature 

Alarm Description
Actual temperature is lower than set low band limit. (Low Band limit = Set temperature - Low band)

Reason & Remedy

1. The temperature of the embossing heater is low (15 0C less than the set temperature).
    a. Check the temperature which is set for the embossing heater.
    b. Check the band value, which is set for embossing the heater.
2.Parameters intered correctly but temperature not risnig.
    a. Check if themocouple is connected to top heater plate properly.
    b. Check if thermocouple and heater supply connector are connected.
    c. Check if the SSR is switching on.
    d. Check if Contactor is on.
    e. Check the heater connection and check voltage during on condition at SSR output and Junction Box.
    f. Check heater resistance.
3. PID parameters.
    a. Check if the deafult PID parameters are shown on machine as per below image.
