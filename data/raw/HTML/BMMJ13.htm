  Error code :BMMJ13 Alarm Message: Sealing Side (L) SW3 Fail

Alarm Description
This alarm occurs when machine is stoped and sealing station side lock cylinder(Left) fails to retun to home position. Side lock cylinder(Left) may not return t home position if cylinder failed to work or if the reed switch position is disturbed or not functioning.

Reason & Remedy

1. Side lock cylinder(Left) falied toreturn to home position.
    a. Check if the sealing side lock cylinder is completly retracted and nothing is blocking it.
    b. Check the pneumatic air pressre. Minimum 4 bar is reqired.
2. Side lock cylinder(Left) retracted but reed switch not sensing the piston.
    a. Check and adjust the position of reed switch 3.
    b. Check if reed switch is tightned properly and not moving while machine is running.
3. Reed switch not sensing.
    a. Check if reed switch is working properly by moving it up and down on clyinder. If reed switch is not functing check the wiring for any loose connection or replace the seed switch.


(Seal Side Lock Left Cylinder)
