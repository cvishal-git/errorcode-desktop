  Error code : BMMJ01 Alarm Message: Guard Open 1- 17 

Alarm Description
Machine guard opened or PLC is not receiving guard protection signal. Check Guard switch input in PLC input card-3.

Reason & Remedy

1.The machine guard is open.
    a. Close guard.
    b. Check whether the guard switch is engaged correctly.
    c. Check electrical wiring.
    d. Check the guard switch for any faults.
    e. Replace the guard switch if needed.
