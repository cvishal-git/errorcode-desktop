  Error code : BMAT03 Alarm Message: Lidding Foil Pin Hole not Ready  

Alarm Description
Lidding Foil Pin is in fault mode or ready signal is not avaliable on machine.

Reason & Remedy

1. Lidding Foil Pin Hole is not Ready/powered on.
    a. Check if the Lidding Foil Pin Hole system is ready and powered on..

2. The ready signal is not coming from the Lidding foil pin hole system.
    a. Check the signal coming from the pin hole system and take corrective action.
