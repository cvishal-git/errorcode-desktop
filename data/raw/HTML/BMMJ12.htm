  Error code :BMMJ12 Alarm Message: Sealing Top unlock SW 1 Fail

Alarm Description
This alarm occurs when machine is stopped and sealing station top plate dose not return to its home position. Sealing station top plate may not return to home position if it is obstructed by side lock cylinder or the top plate cylinder failed to work or if the top reed switch position is disturbed or not functioning.

Reason & Remedy

1. Sealing station failed to unlock after machine stop.
    a. Check if the sealing side lock cylinder is retracted completly and are not blocking top plate from going up.
2. Sealing station opened but reed switch not sensing the piston.
    a. Check and adjust the position of reed switch 1.
    b. Check if reed switch is tightned properly and not moving while machine is running.
3. Reed switch not sensing.
    a. Check if reed switch is working properly by moving it up and down on clyinder. If reed switch is not functing check the wiring for any loose connection or replace the seed switch.
