  Error code : BMAT09 Alarm Message: Stray Product Detected

Alarm Description
Product outside of cavity, product half placed inside cavity, standing tablet, unformed cavity is sensed by stray product sensor.

Reason & Remedy

1. Product is not completed inserted in cavity.
    a. Check the feeding if prodct is placed properly inside cavity.
    b. Check if cavity for poduct is formed properly. Check the depth of forming is as per blister format drawing.
2. Stray prodct sensor not placed correctly.
    a. Check if stray product flap is placed correctly and sensor is sensing onl when start prodct is present.

Open Guard Door 13 to access Stary product sensor.
Note: Ensure machine is stoped before opening the guard door.

(Stary Prodct sensor in off condition/ Stary product is not detected)

(Stary Prodct sensor in on condition/ Stary product is detected)
