  Error code : BMAT02 Alarm Message: Base Foil Pin Hole not Ready 

Alarm Description
Pin Hole is in fault mode or ready signal is not avaliable on machine.

Reason & Remedy

1. Pin Hole is not Ready/powered on.
    a. Check if the Pin Hole system is ready and powered on.

2. The ready signal is not coming from the pin hole system.
    a. Check the signal coming from the pin hole system and take corrective action.
