  Error code : BMAT04 Alarm Message: Pharma Code Error  

Alarm Description
Pharma Code is in fault mode or ready signal is not avaliable on machine.

Reason & Remedy

1. The ready signal is not coming from the Pharma Code system.
    a. Check the signal coming from the Pharma code system and take corrective action.

2. The camera for pharma code is not ready.
    a. Check if the camera is ready for pharma code inspection to start with the camera monitor screen.
