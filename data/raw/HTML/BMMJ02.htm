  Error code : BMMJ02 Alarm Message: PLC SCADA Communiaction Failed 

Alarm Description
PLC communication cable disconnected or PLC is in error mode.

Reason & Remedy

1. The communication between the PLC and the operator panel gets disconnected.
    a. Check that the communication cable is connected correctly.
    b. Check if the PLC and the operator panel communication port are working correctly."
    c. Reset the alarm after appropriate remedies.
