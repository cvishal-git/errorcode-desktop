  Error code : BMAT01 Alarm Message: Camera System not Ready 

Alarm Description
Camera System is in fault mode or ready signal is not avaliable on machine.

Reason & Remedy

1. The camera is not ready.
    a. Check if the camera is ready for blister inspection to start with the camera monitor screen.

2. The signal is not coming from the camera.
    a. Check the signal coming from the camera and take corrective action.

3. The Camera key is off.
    a. On the Camera screen, tap the Camera key to start the function of the camera.
