* Run the machine for 10 cycle and stop. Check the I-mark shifting after 10th stroke. If the shifting for 10 cycle is 2mm then the correction required for every stroke is 0.2mm. Correction value is divided into two parts, basic correction and PRC correction. 60% value should be entered in basic correction and 40% value should be entered in PRC correction. Calculate and enter the value in Basic and PRC correction respectively.
