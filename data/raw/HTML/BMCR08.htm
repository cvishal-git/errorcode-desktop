  Error code : BMCR08 Alarm Message: Forming Top High Temperature 

Alarm Description
Actual temperature is higher than set high band limit.(High Band limit = Set temperature + High band)

Reason & Remedy

1. The temperature of the top heating plate is high (20 0C more than the set temperature).
    a. Check the temperature set for the top heating plate.
    b. Check the band value set for the top heating plate.
2. The heater is on continuously.
    a. Check functioning of Solid State Relay (S.S. R).
    b. Check with multimeter if SSR is giving output in off State. Replace SSR if output is avaliable in off state.
    c. Check if thermocouple is connected properly.
3. PID parameters.
    a. Check if the deafult PID parameters are shown on machine as per below image.
