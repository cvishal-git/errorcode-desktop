  Error code : BMCR16 Alarm Message: Sealing High Temperature 

Alarm Description
Actual temperature is higher than set high band limit.(High Band limit = Set temperature + High band)

Reason & Remedy

1. The temperature of the top sealing plate is high (20 0C more than the set temperature).
    a. Check the temperature set for the top sealing plate.
    b. Check the band value set for the top sealing plate.
    c. Check the temperature sensor of the top sealing plate.
2. Sealing mechanical assembly.
    a. Check the gap between heater and upper sealing plate
    b. Check the hilem sheet between heater and top cooling plate.
    c. Ensure proper water flow.
    d. Check the chiller temperature. Set it as per the format part requirement.
    e. Check if the air blow is connected.
3. The heater is on continuously.
    a. Check functioning of Solid State Relay (S.S. R).
    b. Check with multimeter if SSR is giving output in off State. Replace SSR if output is avaliable in off state.
    c. Check if thermocouple is connected properly.
4. PID parameters.
    a. Check if the deafult PID parameters are shown on machine as per below image.
