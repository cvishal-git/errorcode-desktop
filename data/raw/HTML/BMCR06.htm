  Error code : BMCR06 Alarm Message: Heater Key Off 

Alarm Description
Heater key is turned off form configuration key. High Temperature alarm occured on machine or thermocouple is disconnected.

Reason & Remedy

1. The Heater key is off.
    a. On the Basic screen, tap the Heater key to activate the function of the heater.
2. The chilled water supply is not connected, or the chilled water supply gets switched off.
    a. Check the chilled water supply and take corrective action
3. The set value and the actual value of heaters (forming unit, sealing unit, and perforation unit) are not correct.
    a. Check whether the heater temperature exceeds the set band value limit (forming unit, sealing unit, and perforation unit).
4. Check for the other active alarms
    a. Check if any of the below alarms are active and check there troubleshooting.
