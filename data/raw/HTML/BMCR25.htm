  Error code : BMCR25 Alarm Message: Web Loop Sensor 

Alarm Description
Blister web in indexing station sliped or placed at improper distance.

Reason & Remedy

1. Web loop setting fault.
    a. Web loop should be set between two sensor and sensor should not sense web while machine is running. Refer video tutorial to set webloop.
2. Blister Web Sliped form indexing drum.
    a. Insert the blister web properly inside the indexing drum.
3. Blister web position distrubed form inital position.
    a. Check and adjust the web according to initial position.


(Web Loop Sensor)
4. Web Loop Sensor Faulty.
    a. Check if sensor is high without sensing the web if so, replace the sensor.
