  Error code : BMCR20 Alarm Message: Perforation High Temperature 

Alarm Description
Actual temperature is higher than set high band limit.(High Band limit = Set temperature + High band)

Reason & Remedy

1. The temperature of the perforation heater is high (15 0C less than the set temperature).
    a. Check the temperature which is set for the perforation heater.
    b. Check the band value set for the perforation heater.
    c. Check the temperature sensor of the perforation heater.
2. The heater is on continuously.
    a. Check functioning of Solid State Relay (S.S. R).
    b. heck with multimeter if SSR is giving output in off State. Replace SSR if output is avaliable in off state.
    c. Check if thermocouple is connected properly.
3. PID parameters.
    a. Check if the deafult PID parameters are shown on machine as per below image.
