  Error code : BMMI01 Alarm Message: Base Foil 1 Low Level 

Alarm Description
Base foil is less than set level.

Reason & Remedy

1. The forming film gets low on the forming film mounting unit.
    a. Insert new forming film reel on forming film mounting unit.
2. The sensor is not working correctly.
    a. Check the sensor and set it correctly. Refer Chapter "Maintenance" > Section " Setting the Diffuse Reflection Sensor"
