  Error code : BMMJ10 Alarm Message: Indexing Fault 

Alarm Description
This alarm occurs if Web is slipped form indexing drum and indexing sensor is sensed.

Reason & Remedy

1. The indexing sensor gets on due to web slippage from the indexing roller.
    a. Check web slippage from the indexing roller and insert it properly into the punching unit.
    b. Check if the indexing sensor gets on by touching the object.
