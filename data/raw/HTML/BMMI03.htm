  Error code : BMMI03 Alarm Message: Lidding Foil Low Level 

Alarm Description
Lidding foil is less than the set level.

Reason & Remedy

1. The lidding foil gets low on the lidding foil mounting unit
    a. Insert a new lidding foil reel on the lidding foil mounting unit.
    b. Check the sensor and set it correctly. Refer Chapter "Maintenance" > Section "Setting the Diffuse Reflection Sensor"
2. The sensor is not working correctly.
    a. Replace the sensor if it is faulty.
