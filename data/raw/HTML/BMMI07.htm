  Error code : BMMI07 Alarm Message: Overflow Blister Bin Full

Alarm Description
Overflow blister collection bin full

Reason & Remedy

1.Overflow Blister Bin is Full.
    a. Empty the Overflow Blister Bin.
2. Overflow Blister Bin sensor blocked by Blister or other objest.
    a. Clear the Blister or the object blocking the Overflow Blister Bin sensor.
3. Sensor is faulty.
    a. Dismount the sensor and check if sensor is giving false output. If so replace the sensor.
