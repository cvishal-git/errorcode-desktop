  Error code : BMCR07 Alarm Message: Forming Top Low Temperature 

Alarm Description
Actual temperature is lower than set low band limit. (Low Band limit = Set temperature - Low band)

Reason & Remedy

1. The temperature of the top heating plate is low (15 0C less than the set temperature).
    a. Check the temperature set for the top heating plate.
    b. Check the band value set for the top heating plate.
2. Parameters are OK but temperature not risnig.
    a. Check if themocouple is connected to top heater plate properly.
    b. Check if the SSR is switching on.
    c. Check if Contactor is on.
    d. Check the heater connection and check voltage during on condition at SSR output and Junction Box.
    e. Check heater resistance.
3. PID parameters.
    a. Check if the deafult PID parameters are shown on machine as per below image.
