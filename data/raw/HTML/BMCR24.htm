  Error code : BMCR24 Alarm Message: Embossing Thermocouple Open  

Alarm Description
Thermocouple disconneced form temperature card or at connector.

Reason & Remedy

1. The temperature sensor is not connected to the embossing heater.
    a. Connect temperature sensor to embossing heater.
    b. Check if the thermocouple cable is open or damaged. Check the Thermocouple connector and cable for continuty. Replace cable if continuity not found.
2. The temperature sensor failed.
    a. Replace the temperature sensor.

  Thermocouple Connector

  Thermocouple Connector Opened

  Thermocouple Connector Closed
